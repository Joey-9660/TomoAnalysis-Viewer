Module 1: PDD Framework 개요

1.1 PDD Framework란?

  • 정의: Holotomography 기반 세포 표현형(Phenotype) 관찰을 약물 반응 의사결정(Decision)으로 연결하는 체계적 방법론
  • 핵심 가치: Label-free, 비침습적, 정량적, 실시간 모니터링 가능
  • 차별점: 단일 파라미터 endpoint assay 대비 다차원 표현형 프로파일링 제공

1.2 Framework 구성 요소

Drug-Phenotype Mapping Table

8개 약물 클래스 → 23개 표현형 매핑

Phenotype Descriptor Library

23개 세포 표현형 정의 및 정량 기준

Core Metric Library

22개 정량 지표 및 측정 방법

1.3 대상 Therapeutic Areas

Oncology Immuno-Oncology Immunology Hepatotoxicity Hematology

Module 2: PDD 워크플로우 (5단계)

1

Data Acquisition (데이터 획득)

Tomocube 시스템으로 3D HT/FL 데이터 획득. Time-lapse 프로토콜 설정.

실습: HT-X1 기본 촬영 및 time-lapse 설정

2

Phenotype Identification (표현형 식별)

TomoAnalysis Viewer에서 데이터 시각화 및 Phenotype Library 기반 표현형 식별.

실습: 2D/3D/Ortho View 활용, 약물 처리 전후 비교

3

Metric Quantification (지표 정량화)

Core Metric Library 기반 표현형 정량화. Viewer 내장 분석 도구 활용.

실습: Histogram, Intensity Plot, Line Measurement 활용

4

Drug-Phenotype Mapping (약물-표현형 매핑)

관찰된 표현형 프로파일을 Mapping Table과 대조. 패턴 매칭으로 MoA 분류.

실습: Cisplatin 처리 세포의 DPM-001 패턴 매핑

5

Decision Support (의사결정 지원)

Efficacy, Resistance, Toxicity 판정 기준 적용. Application Note 작성.

실습: Decision criteria 적용 및 보고서 작성

Module 3: 주요 표현형 해설

고객 교육 시 가장 빈번하게 활용되는 핵심 표현형:

PHE-001: Cell Volume Increase (세포 부피 증가)

측정: MET-001 (Cell Volume) | 도구: 3D View + Volume Measurement

기준: Mild (10-20%), Moderate (20-40%), Severe (>40%)

PHE-002: Nuclear Condensation (핵 응축)

측정: MET-002, MET-006 | 도구: Histogram + 3D View

기준: Nuclear RI 증가 0.002-0.005 (Mild), 0.005-0.01 (Moderate), >0.01 (Severe)

PHE-008: Dry Mass (건조질량)

측정: MET-008 (Dry Mass) | 도구: Intensity Plot + Area Plot

기준: 감소 5-15% (Mild), 15-30% (Moderate), >30% (Severe)

PHE-021: Lipid Droplet Accumulation (지질 방울 축적)

측정: MET-021 (Area Fraction) | 도구: Histogram + 3D View

기준: Normal (<5%), Mild (5-15%), Moderate (15-30%), Severe (>30%)

Module 4: Use Case 실습

실습 1: 항암제 효능 평가 (Oncology Drug Screening)

시나리오: Cisplatin 처리된 A549 폐암 세포의 반응 평가

참조 매핑: DPM-001 (Alkylating Agent)

확인 표현형: PHE-001 (Swelling), PHE-002 (Nuclear Condensation), PHE-003 (Blebbing)

정량 지표: MET-001 (Volume), MET-003 (Mean RI), MET-008 (Dry Mass), MET-017 (Death Rate)

의사결정: Volume 변화 >20% 및 RI shift >0.005 → 유효성 확인

실습 2: 간독성 안전성 평가 (Hepatotoxicity Screening)

시나리오: Acetaminophen 고농도 처리된 hepatocyte의 독성 평가

참조 매핑: DPM-008 (Hepatotoxicity Inducer)

확인 표현형: PHE-021 (Lipid Droplets), PHE-022 (Swelling), PHE-010 (RI Increase)

정량 지표: MET-021 (Lipid Area Fraction), MET-001 (Volume), MET-003 (Mean RI)

의사결정: Dose-dependent lipid 축적 및 RI 증가 >0.008 → 간독성 위험 경고

실습 3: 면역항암제 효능 평가 (Immunotherapy Assessment)

시나리오: Anti-PD-1 처리된 면역세포-종양세포 공배양 시스템

참조 매핑: DPM-004 (Checkpoint Inhibitor)

확인 표현형: PHE-011 (T-cell Activation), PHE-012 (Clustering), PHE-013 (Tumor Lysis)

정량 지표: MET-015 (Clustering Index), MET-016 (Contact Ratio), MET-017 (Death Rate)

의사결정: T-cell volume 증가 + 종양 주변 clustering → 면역 활성화 확인
